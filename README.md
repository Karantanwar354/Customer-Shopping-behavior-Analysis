# Customer-Shopping-behavior-Analysis
Analyzed retail consumer shopping data using Python, SQL, and Power BI to identify trends across customer demographics, product categories, and online and offline sales channels. Performed data cleaning, exploratory analysis, and built an interactive dashboard to uncover purchase drivers, customer segments, and loyalty patterns.

## Overview

This project focuses on analyzing retail consumer shopping data to understand customer behavior and identify trends that support data-driven business decisions. The analysis covers data loading, cleaning, exploratory analysis, SQL-based querying, visualization, and reporting using industry-standard tools.

---

## Dataset

The dataset contains retail consumer shopping information, including customer demographics, product categories, sales channels (online and offline), purchase details, and payment preferences.
It is used to analyze purchasing patterns, customer segments, and factors influencing repeat purchases.

## Analysis Workflow

1. Loaded the dataset into Python and performed initial data inspection.
2. Cleaned and transformed raw data by handling missing values, duplicates, and inconsistent formats.
3. Conducted exploratory data analysis (EDA) to identify trends and patterns.
4. Imported cleaned data into PostgreSQL and executed SQL queries to analyze customer segments, loyalty, and purchase drivers.
5. Built an interactive Power BI dashboard to visualize insights.
6. Summarized findings and recommendations in a project report.

---

## Dashboard

The Power BI dashboard highlights:

* Customer demographics and segmentation
* Sales trends across product categories
* Online vs. offline channel performance
* Loyalty and repeat purchase behavior

The dashboard enables stakeholders to interact with the data and make informed decisions.

---

## Results & Insights

* Identified key purchasing trends across demographics and sales channels.
* Analyzed the impact of discounts, seasons, and payment preferences on buying behavior.
* Highlighted customer segments with higher repeat purchase potential.

---

## How to Run the Project

1. Clone the repository.
2. Open the Jupyter Notebook and run the Python scripts for data cleaning and EDA.
3. Load the cleaned dataset into PostgreSQL and execute the provided SQL queries.
4. Open the Power BI file to explore the interactive dashboard.
5. Refer to the report for summarized insights and business recommendations.
